    Upcoming events sponsored by UDIP!

    Develop an efficient coding workflow that prepares you for physics research, graduate school, or industry. In six 90-minute sessions, the PHS will teach productive programming practices in an informal setting, and produce a streamlined computing environment for file system navigation (bash), text editing (vim), numerical simulations (python), and report typesetting (LaTeX). All operating systems welcome (if using Windows 10, install Windows Subsystem for Linux before the meeting). All physics undergraduates welcome, especially freshmen/sophomore students that want to learn computing tools relevant for research.

    Want a sneak peek of material, miss a week's lesson, or want to follow along on your own? All PHS guides are self-contained and freely available at github.com/erijones/phs.
